EOD Stock API - EOD Documentation

Description: This endpoint returns end-of-day data for a certain stock on a specific date.

Endpoint : https://gateway.eod-stock-api.site/api/v1/eod/{_date}/{exchange_code}?api_key=YOUR_API_KEY

HTTP Method: GET

Parameters :

  • _date (required): The date for which to retrieve the end-of-day data. This should be in the format yyyy-mm-dd.
  • exchange_code (required): The code for the exchange on which the stock is listed. This should be a string.

  • Python
  • Node
  • Response
This function takes in an api_key, a date in the format yyyy-mm-dd, and an exchange_code as arguments, and returns the response data from the API in the form of a dictionary. If the API returns an error status code, it raises an HTTPError. You can use this function to retrieve end-of-day data for a certain stock on a specific date. 

import requests

def get_eod_data(api_key, date, exchange_code):
    """
    Makes a GET request to the end-of-day data endpoint and returns the response data.

    Args:
    api_key (str): Your API key.
    date (str): The date for which to retrieve the end-of-day data in the format yyyy-mm-dd.
    exchange_code (str): The code for the exchange on which the stock is listed.

    Returns:
    dict: The response data from the endpoint.

    Raises:
    requests.exceptions.HTTPError: If the API returns an error status code.
    """
    url = f"https://gateway.eod-stock-api.site/api/v1/eod/{date}/{exchange_code}?api_key={api_key}"
    with requests.Session() as session:
        response = session.get(url)
        response.raise_for_status() # Raise an HTTPError if status code is unsuccessful
    return response.json()

This function takes in an apiKey, a date in the format yyyy-mm-dd, and an exchangeCode as arguments, and returns a promise that resolves with the response data from the API. If the API returns an error status code, it throws an error with a message indicating the reason for the error.

const axios = require('axios');

async function getEodData(apiKey, date, exchangeCode) {
  const url = `https://gateway.eod-stock-api.site/api/v1/eod/${date}/${exchangeCode}?api_key=${apiKey}`;
  try {
    const response = await axios.get(url);
    return response.data;
  } catch (error) {
    throw new Error(`Error retrieving end-of-day data: ${error.message}`);
  }
}
