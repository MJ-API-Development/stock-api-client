EOD Stock API - Documentations - Financial News

API Endpoint

The API endpoint for retrieving financial news is https://eodhistoricaldata.com/api/news.

Parameters

The financial news API endpoint accepts the following parameters:

  • symbols (required) - A comma-separated list of stock symbols to retrieve news for.
  • from (optional) - A date string in the format YYYY-MM-DD specifying the start of the date range to retrieve news for.
  • to (optional) - A date string in the format YYYY-MM-DD specifying the end of the date range to retrieve news for.
  • sources (optional) - A comma-separated list of news sources to retrieve news from.
Response

The financial news API endpoint returns an array of news articles, where each article is represented as a JSON object with the following fields:

  • id - The unique identifier of the news article.
  • title - The title of the news article.
  • description - A short description of the news article.
  • url - The URL of the news article.
  • source - The source of the news article.
  • date - The date the news article was published.
