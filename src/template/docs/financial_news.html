EOD Stock API - Financial News API Docs

Endpoint: https://gateway.eod-stock-api.site/api/v1/news/articles-by-ticker/{stock_code}?api_key=YOUR-API-KEY

Description: This endpoint retrieves financial news articles related to a specific stock code.

HTTP Method: GET

URL Parameters:

  • - {stock_code} - the code of the stock to retrieve the news articles for.

You can obtain your API Key once you subscribe

  • Python
  • Node
  • Response

Response format:

The API will return a JSON object with the following properties:

  • - message: A string describing the status of the request.
  • - payload: An array of news articles related to the stock code. Each article is represented as a JSON object with the following properties:
  • - link: The URL link to the full article.
  • - providerPublishTime: The time the article was published, represented as a Unix timestamp.
  • - publisher: The publisher of the article.
  • - relatedTickers: A JSON object containing related stock tickers for the article.
  • - thumbnail: A JSON object containing information about the thumbnail image for the article.
  • - title: The title of the article.
  • - type: The type of the article.
  • - uuid: The unique identifier for the article.
  • - status: A boolean indicating the status of the request.

Example Request:

GET https://gateway.eod-stock-api.site/api/v1/news/articles-by-ticker/GOOG?api_key=YOURAPIKEY
