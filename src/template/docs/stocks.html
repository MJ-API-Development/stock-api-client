EOD Stock API - Stocks Endpoints

The Stocks endpoints allow you to retrieve data related to stocks. Currently, there are several endpoints available:

Single Stock

Get details for a specific stock given its code.

Endpoint URL: /api/v1/stock/code/{stock_code}

Params: stock_code (string) - the code of the stock to retrieve details for.

Try it out
All Stocks

Get a complete list of all stocks available.

Endpoint URL: /api/v1/stocks

Params: None

Try it out
Stocks by Country

Get a list of stocks listed in a certain country.

Endpoint URL: /api/v1/stocks/country/{country}

Input: country (string) - the name of the country to retrieve stocks for.

Try it out
Stocks by Currency

Get a list of stocks traded in a certain currency.

Try it out
Stocks by Exchange Code

Get a list of stocks listed on a certain exchange given its code.
