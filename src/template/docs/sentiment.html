EOD Stock API - Sentiment Analysis Endpoints

The Sentiment Analysis endpoints allow you to retrieve sentiment-related data for a specific stock. Currently, there are three endpoints available:

Trend Setters

Get a list of Twitter handles that are currently driving sentiment for a given stock.

GET /api/v1/sentiment/trend-setters/stock/{stock_code}

The {stock_code} parameter in the endpoint should be replaced with the stock's symbol, such as AAPL for Apple Inc.

Example Response:

[
    {
        "user_handle": "@elonmusk",
        "score": 0.85
    },
    {
        "user_handle": "@CathieDWood",
        "score": 0.7
    }
]
Trending Tweets

Get a list of the top tweets that are currently driving sentiment for a given stock.

GET /api/v1/sentiment/trending/stock/{stock_code}

The {stock_code} parameter in the endpoint should be replaced with the stock's symbol, such as AAPL for Apple Inc.

Example Response:

[
    {
        "text": "Great news for Tesla today!",
        "user_handle": "@elonmusk",
        "score": 0.8
    },
    {
        "text": "Just bought some AAPL shares, excited to see where this goes.",
        "user_handle": "@investor123",
        "score": 0.5
    }
]
Single Tweet

Get sentiment analysis for a single tweet related to a given stock.

GET /api/v1/sentiment/tweet/stock/{stock_code}

The {stock_code} parameter in the endpoint should be replaced with the stock's symbol, such as AAPL for Apple Inc.

Example Response:

{
    "text": "Great news for Tesla today!",
    "user_handle": "@elonmusk",
    "score": 0.8
}
