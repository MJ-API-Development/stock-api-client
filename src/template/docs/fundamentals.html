Fundamentals Data Documentation

This API provides fundamental data for companies based on their stock code. The endpoint for accessing the data is:

Endpoint

https://gateway.eod-stock-api.site/api/v1/fundamental/company/{stock_code}

Using Python Requests

You can easily use python requests to access our api here is a simplified example below


"""
Retrieve information about a company identified by its stock code from the EOD Stock API.

:param stock_code: The stock code of the company to retrieve information for.
:type stock_code: str

:param api_key: The API key to use to access the EOD Stock API.
:type api_key: str

:return: A dictionary containing information about the company, or None if an error occurred.
:rtype: dict or None
"""

import requests

def get_company_info(stock_code: str, api_key: str) -> dict or None:
    """
    Retrieve information about a company identified by its stock code from the EOD Stock API.

    :param stock_code: The stock code of the company to retrieve information for.
    :type stock_code: str

    :param api_key: The API key to use to access the EOD Stock API.
    :type api_key: str

    :return: A dictionary containing information about the company, or None if an error occurred.
    :rtype: dict or None
    """

    url = f"https://gateway.eod-stock-api.site/api/v1/fundamental/company/{stock_code}"
    params = {"api_key": api_key}
    response = requests.get(url, params=params)

    if response.status_code == 200:
        return response.json()
    else:
        print(f"Error {response.status_code}: {response.reason}")
        return None

What is Company Fundamental Data

When analyzing a company, one of the most important aspects to consider is its fundamental data. Fundamental data refers to the financial and economic information that is used to evaluate a company's performance and determine its investment potential. This information can be found in a company's financial statements, which are required by law to be published periodically.

Income Statement

This statement shows a company's revenue, expenses, and profit over a specific period of time. It provides insight into how the company is performing financially, including its profitability and revenue growth.

Balance Sheet

The balance sheet shows a company's assets, liabilities, and equity at a specific point in time. This statement provides information about the company's financial health, including its liquidity and solvency.

Cash Flow Statement

This statement shows the inflows and outflows of cash from a company's operations, investments, and financing activities. It provides information about a company's ability to generate cash, pay dividends, and fund capital expenditures.

Key Financial Ratios

These ratios are used to analyze a company's financial performance relative to its peers and the industry as a whole. Examples include the price-to-earnings ratio, return on equity, and debt-to-equity ratio.

Economic Data

This includes macroeconomic indicators such as GDP, inflation, and interest rates. These factors can impact a company's performance, so it's important to consider them when analyzing fundamental data.

Fundamental data is used by investors and analysts to determine a company's value and potential for growth. By analyzing financial statements and economic data, they can make informed decisions about buying, holding, or selling a company's stock.

It's important to note that fundamental data is just one piece of the puzzle when it comes to investing. Other factors, such as market trends, political events, and company-specific news, can also impact a stock's performance. Therefore, it's important to conduct thorough research and analysis before making any investment decisions.

In conclusion, understanding company fundamental data is crucial for anyone looking to invest in the stock market. By analyzing a company's financial statements and economic data, investors can make informed decisions about the company's potential for growth and profitability.
